📊 Analyse Exploratoire de Données (EDA) - Marché du Jeu Vidéo
Projet Personnel - Python, Pandas, Seaborn

🎯 Objectif
Ce projet a pour but d'analyser un dataset contenant les ventes mondiales de jeux vidéo (plus de 16 000 entrées) afin d'en extraire des tendances clés.

🛠️ Compétences techniques démontrées
Data Cleaning : Nettoyage des valeurs nulles et formatage du dataset avec Pandas.

Data Visualization : Création de graphiques statistiques avec Matplotlib et Seaborn pour interpréter les volumes de ventes par plateforme et par genre.
