# 📊 Analyse des Ventes de Jeux Vidéo (Python)

## 🎯 Objectif du Projet
Ce projet analyse un dataset de 16 500 jeux vidéo pour identifier les tendances du marché (plateformes les plus populaires et genres les plus vendus).

## 🛠️ Compétences Techniques
* **Langage :** Python
* **Librairies :** * `Pandas` : Manipulation et nettoyage des données (Data Cleaning).
  * `NumPy` : Calculs statistiques (record de ventes, moyennes).
  * `Seaborn` & `Matplotlib` : Visualisation de données.

## 📈 Résultats obtenus
Le script génère automatiquement des analyses visuelles :
- Top 10 des consoles avec le plus de jeux.
- Volume des ventes mondiales par genre.
